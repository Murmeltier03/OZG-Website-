# OZG Click-Dummy: Bewohnerparkausweis

Diese Demo ist eine lokale, statische Click-Website fuer einen englischsprachigen Workshop zum Onlinezugangsgesetz.

## Start

Oeffne `index.html` direkt im Browser. Es ist keine Installation und kein lokaler Server notwendig.

## Bewusst eingebaute Fehler

- Fehler 1: Schlechte Auffindbarkeit durch kaputte Suche und versteckten Navigationspfad.
- Fehler 2: Medienbruch am Ende durch Drucken, Unterschreiben und Postversand.
- Fehler 3: Mehrfacheingabe bereits bekannter Personen-, Adress- und Fahrzeugdaten.
- Fehler 8: Kein Online-Payment, nur spaetere Ueberweisung.
- Fehler 10: Formular-Download statt echter Online-Antragstellung.

Beim Start erscheint das Szenario als Popup ueber einem unscharfen Portal-Hintergrund. Die rechte Seitenleiste zeigt alle zehn Fehler als aufklappbare Informationsliste, ohne in der Website zu verraten, welche davon im Klickpfad eingebaut sind.
